Understanding the Basics of CPU

Memory, Motherboards, and Power Supplies

1. What are the two basic components inside a CPU?

A classic CPU has two basic components. First is the arithmetic logic unit also known as (ALU), which performs tasks such as arithmetic and logical operations, second is the control unit also known as CU which manages various components of the computer (Bialowas, 2023).

2. What is the difference between static and dynamic memory?

Static memory allocation, in static memory allocation the size of the memory, is determined by the time it takes to compile meaning it cannot be changed while the program is still running, It is also limited in flexibility the file itself cannot be altered once its allocated, it's better use for arrays or data structures.

While dynamic memory is known as flexible as it can be allocated at runtime meaning it can be adjusted when needed even if the program is still running. However, static memory is simpler but more rigid, while dynamic memory is more flexible but requires careful management to avoid issues like memory leaks (UNacademy, 2022).

3. What are four ways in which one motherboard may differ from another?

1. Brand and version, depending on the motherboards brand and version some motherboards can outperform other motherboard brands especially if it's old.

2. Sata sockets, some motherboards have only few SATA sockets, some have many and it differs whether your unit can handle multiple storage devices HDD/SDD or even M.2.

3. Design, some motherboards are built to be more compact and not take a big chunk of space inside the unit.

4. RAM slots, some motherboards can fit more than four RAM slots, some can only fit two RAM inside the motherboard. And these are the four ways in which one motherboard may differ from other; some can be defined in design, some in model and some for its compatibility (UNacademy, 2022).

4. Why does a power supply have different colored wires?

The wires are color-coded based on the types of electric flow through the wires, type of current, phases of the wires, and on the basis or order of connection (Paramount n.d).
